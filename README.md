# 🎓 BunkBuddy
**Plan Smart • Bunk Smarter 😎**

---

## 📌 Project Description
BunkBuddy is a smart attendance planning web application designed to help students monitor and manage their attendance efficiently. The system allows users to select their branch, semester, and subject to calculate attendance status and determine whether they can safely bunk upcoming classes while maintaining the required attendance percentage.

---

## 🚀 Tech Stack
- HTML5
- CSS3 (Glassmorphism UI + animations)
- JavaScript (logic & dynamic UI)
- JSON (timetable & subject data)

---

## ✨ Features
- Branch and semester selection
- Subject-wise attendance calculation
- Smart bunk prediction logic
- Responsive glassmorphism UI
- Floating gradient animated background
- Timetable-based subject mapping
- Attendance status display with strategy suggestion

---

## ⚙ Installation Commands
```bash
git clone https://github.com/your-username/BunkBuddy-.git
cd BunkBuddy-
```

## ▶ Run Commands
This is a static frontend application.

Open the project by launching index.html in any browser.

Alternatively, run a local server:

```bash
python -m http.server
```
